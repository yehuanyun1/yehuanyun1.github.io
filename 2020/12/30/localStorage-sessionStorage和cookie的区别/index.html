localStorage,sessionStorage和cookie的区别
Keep team Lv2
2020-12-30 08:40:54

HTML5 Web 存储是本地存储，存储在客户端，包括 localStorage 和 sessionStorage。HTML5 Web 存储是以键/值对的形式存储的，通常以字符串存储。

localStorage

localStorage 生命周期是永久，除非主动清除 localStorage 信息，否则这些信息将永远存在。存放数据大小为一般为 5MB,而且它仅在客户端（即浏览器）中保存，不参与和服务器的通信。

sessionStorage

sessionStorage 仅在当前会话下有效，关闭页面或浏览器后被清除。存放数据大小为一般为 5MB,而且它仅在客户端（即浏览器）中保存，不参与和服务器的通信。
